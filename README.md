##My Coffee App API

### Used Technologies
1. Express Js
2. GraphQL Apollo Server
3. Mongo DB
4. DB Seeder
5. Nodemon inspect
6. ESLint


**.env.locale**
```bash
MONGO_URI={your Mongo URL}
API_URL={your API URL}
PORT={your default PORT}
JWT_SECRET={Example: 7bc78545b1a3923cc1e1e19523fd5c3f20b409509}
BASE_EMAIL={Example: mycoffee@gmail.com}
````
## Available Scripts
